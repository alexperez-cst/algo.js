# asap-algo.js
## As Simple As Possible Javascript Library with the implementations of all the algorithms that i'm learning.
### Download
```
npm install --save asap-algo.js
```

## Functions:
### Array Searching
* binarySearch O(log(n))
### Array Sort
* bubbleSort O(n^2)
* insertionSort O(n^2)
* selectionSort O(n^2)
* mergeSort O(n * log(n))
* quickSort O(n * log(n))
